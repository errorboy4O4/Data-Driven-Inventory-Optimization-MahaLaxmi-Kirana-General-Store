# Data-Driven-Inventory-Optimization-MahaLaxmi-Kirana-General-Store

Analyzed 4,500+ real retail transactions and inventory records collected from handwritten store registers to identify stockouts, wastage drivers, and demand patterns across 1,200+ SKUs. 

Applied ABC classification, time-series analysis, and category-wise profit analysis using Excel and Python to optimize reordering decisions and reduce perishables wastage. 

Delivered actionable recommendations that improved inventory prioritization, reduced stockouts in high-demand items, and supported profit optimization through high-margin product focus.
